# Caçamba CE - Fortaleza

Site institucional desenvolvido para serviços de aluguel de caçambas e containers de entulho na região de Fortaleza, CE.

## Tecnologias Utilizadas
* HTML5
* CSS (Interno)
* JavaScript (Interno para painel/toast)

## Páginas
* Início (index.html)
* Aluguel de Caçamba
* Aluguel de Container
* Disk Entulho
* Política de Privacidade
* Termos de Us
